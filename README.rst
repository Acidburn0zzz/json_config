Getting Started
---------------

Installation
~~~~~~~~~~~~

At the command line either via easy_install or pip::

    $ easy_install json_config
    $ pip install json_config

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv json_config
    $ pip install json_config

Usage
~~~~~
To use json_config in a project::

    import json_config
